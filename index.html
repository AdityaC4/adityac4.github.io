current work

contributing to LLVM/Clang, focusing on constexpr evaluation and SIMD optimization. writing about the experience.


research highlights

  • when does the compiler pass ordering matter? controlled experiments on a WebAssembly compiler reveal small global effects (0.25%) but systematic sensitivity in nested loops (3-5% mean, outliers to 10%), motivating per-program selection. blog & report
  • upstreamed constexpr support for x86 vector intrinsics to LLVM: PR 158778 and PR 161302.
  • authored the IO500-based GPFS file system analysis of MSU HPCC storage; report
  • evaluated campus RFID access control security (MIFARE Classic and HID/iCLASS); findings documented in reports.
